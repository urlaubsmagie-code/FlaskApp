2. Create migrate instance at module level (after imports):
   ```python

   ```

3. In create_app(), after db.init_app(app), add migrate initialization:
   ```python
   migrate.init_app(app, db, render_as_batch=True)
   ```
   Note: render_as_batch=True is default in Flask-Migrate 4.0+ but explicit is clearer.

4. Add WAL mode setup function BEFORE create_app():
   ```python
   def _setup_sqlite_pragmas(engine):
       """Configure SQLite connection pragmas for WAL mode and busy timeout."""
       @event.listens_for(engine, "connect")
       def set_sqlite_pragma(dbapi_connection, connection_record):
           cursor = dbapi_connection.cursor()
           cursor.execute("PRAGMA journal_mode=WAL")
           cursor.execute("PRAGMA busy_timeout=5000")
           cursor.execute("PRAGMA foreign_keys=ON")
           cursor.close()
   ```

5. In create_app(), after db.init_app(app), call WAL setup if using SQLite:
   ```python
   # Enable WAL mode for SQLite (concurrent access for polling)
   if 'sqlite' in app.config.get('SQLALCHEMY_DATABASE_URI', ''):
       _setup_sqlite_pragmas(db.engine)
   ```
