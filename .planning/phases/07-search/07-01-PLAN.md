  <action>
Add `/api/search` GET endpoint to routes.py that:

1. Import search utilities at top of route function:
   ```python
   from .utils.search import search_messages, get_search_snippet
   ```

2. Accept query parameters:
   - `q` (required): Search query string
   - `platform` (optional): Filter results by platform
   - `status` (optional): Filter results by status

3. Handle empty query: Return `{'results': [], 'query': '', 'total': 0}`

4. Call `search_messages(query, limit=100)` to get FTS5 results

5. For each result, call `get_search_snippet(query, result['message_id'])` to get highlighted snippet

6. Sanitize snippets to prevent XSS while allowing `<mark>` tags:
   ```python
   import html
   def sanitize_snippet(snippet):
       if not snippet:
           return None
       escaped = html.escape(snippet)
       escaped = escaped.replace('&lt;mark&gt;', '<mark>')
       escaped = escaped.replace('&lt;/mark&gt;', '</mark>')
       return escaped
   ```

7. Optionally filter results by platform/status if those params provided

8. Group results by conversation_id:
   ```python
   grouped = {}
   for r in results:
       conv_id = r['conversation_id']
       if conv_id not in grouped:
           grouped[conv_id] = {
               'conversation_id': conv_id,
               'guest_name': r['guest_name'],
               'guest_id': r['guest_id'],
               'subject': r['subject'],
               'platform': r['platform'],
               'match_count': 0,
               'first_snippet': None
           }
       grouped[conv_id]['match_count'] += 1
       if grouped[conv_id]['first_snippet'] is None:
           grouped[conv_id]['first_snippet'] = r.get('snippet')
   ```

9. Return JSON: `{'results': list(grouped.values()), 'query': query, 'total': len(grouped)}`

Place this endpoint near other API routes (around line 90, after api_get_conversations).
